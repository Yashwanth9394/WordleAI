## 📱 App Structure Strategy

### Option A: Single App, Multiple Domains
```
App Name: "Quivio - Daily Brain Games"

Inside app:
┌─────────────────────┐
│   Choose Domain     │
├─────────────────────┤
│ 🔤 Vocabulary       │
│ 🌍 Geography        │
│ 🧪 Science          │
│ 💻 Tech             │
│ 🎨 Pop Culture      │
│ 📚 History          │
│ 👶 Kids Mode        │
└─────────────────────┘

Free: 1 domain
Premium: All domains
```

**Pros:**
- One app to maintain
- Cross-promotion between domains
- Easier user management
- Unified statistics

**Cons:**
- Larger app size
- Harder to rank for specific categories

---

### Option B: Separate Apps per Major Category
```
Main app: "Quivio"
Kids app: "Quivio Kids"
Pro app: "Quivio Pro" (for professionals - tech, science)

Cross-promotion between apps
```

**Pros:**
- Targeted to specific audiences
- Smaller app sizes
- Better App Store rankings per category
- Kids app can be truly kid-safe

**Cons:**
- More maintenance
- Harder to track users across apps
